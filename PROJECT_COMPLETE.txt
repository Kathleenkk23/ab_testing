PROJECT SUMMARY
===============

A production-ready A/B testing platform with ML-based variant assignment using
an epsilon-greedy bandit algorithm. Built with FastAPI, SQLite, and SQLAlchemy.

======================

   • FastAPI REST API with 6 endpoints + documentation
   • SQLite database with 3 models (Experiment, Assignment, Event)

   • Pydantic validation for all requests
   • CORS middleware for cross-origin requests
   • Auto-generated Swagger UI at /docs
